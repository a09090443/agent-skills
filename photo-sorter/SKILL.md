# Photo Sorter Skill

## 描述 (Description)
這個技能可以幫助使用者將含有 EXIF GPS 座標資訊的相片，自動依照「國家_城市_區」的格式建立資料夾並進行分類。同時它具備以下進階功能：
1. **1 公里範圍合併**：若新相片的位置與已知地點距離小於 1 公里，會自動歸類到同一個資料夾。
2. **日期重新命名**：進入資料夾的相片會自動用相片的「拍攝日期與時間」作為新檔名（例如：`2023-01-01_12-00-00.jpg`）。
3. **無定位相片處理**：若相片沒有 GPS 資訊，會統一移至「其他」資料夾，並在裡面依據「日期」建立子資料夾存放（例如：`其他/2023-01-01/`）。

## 何時使用 (When to use)
- 當使用者要求「依據位置分類相片」、「整理 Google 相簿下載的照片」、「依照座標整理圖片」時。
- 當使用者提到「EXIF」、「GPS座標」、「相片分類」等關鍵字時。

## 執行步驟 (Execution)
1. **確認路徑**：確認使用者有提供「輸入資料夾（包含未分類相片）」與「輸出資料夾（預計存放分類後相片）」的絕對路徑。若未提供，請主動向使用者詢問。
2. **準備環境**：確保環境已安裝必要套件。若需安裝，請執行 `pip install geopy Pillow`。
3. **執行腳本**：使用 run_command 執行腳本：
   `python .agent/skills/photo-sorter/scripts/sort_photos.py -i "<輸入路徑>" -o "<輸出路徑>"`
4. **回報結果**：觀察終端機輸出，並向使用者回報分類結果與產生的狀態。
