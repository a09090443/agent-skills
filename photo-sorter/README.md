# Photo Sorter (相片分類工具)

這是一個自訂的 Agent Skill，主要用來幫助使用者將含有 EXIF GPS 座標資訊的相片，自動依照「國家_城市_區」的格式建立資料夾並進行分類。

## 主要功能

1. **地理位置分類**：自動解析相片 EXIF 中的 GPS 資訊，並透過逆向地理編碼轉換為「國家_城市_區」的資料夾結構。
2. **1 公里範圍合併**：若新相片的位置與已知地點距離小於 1 公里，會自動歸類到同一個資料夾，避免產生過多零碎的目錄。
3. **日期重新命名**：進入分類資料夾的相片，會自動使用相片的「拍攝日期與時間」作為新檔名（例如：`2023-01-01_12-00-00.jpg`）。
4. **無定位相片處理**：若相片沒有包含 GPS 資訊，會統一移至「其他」資料夾，並在裡面依據「拍攝日期」建立子資料夾存放（例如：`其他/2023-01-01/`）。

## 如何觸發使用 (How to trigger)

若你已經正確設定了 Agent Skills 的環境（參考主目錄的 README），在與 AI 對話時只要提及以下 **關鍵字** 或 **需求情境**，AI 助理便會自動去尋找並觸發本技能：

- **情境指令**：「依據位置分類相片」、「整理 Google 相簿下載的照片」、「依照座標整理圖片」
- **觸發關鍵字**：「EXIF」、「GPS座標」、「相片分類」

**💡 範例提示詞 (Prompt)：**
> 「請整理我剛剛下載的 Google 相簿照片，未分類的照片放在 `C:\Downloads\Photos`，請將分類好的照片輸出到 `D:\Organized_Photos`。」
